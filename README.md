# ROFI Wi-Fi Menu Script

A simple script to display and manage Wi-Fi networks using a ROFI menu interface.

Features:
- List nearby Wi-Fi networks in a ROFI menu
- Connect to a selected network using NetworkManager
- Lightweight and suitable for minimal Linux setups

The script is designed to be easy to modify and integrate into custom rice configurations.

This scripts improves upon the bugs of https://github.com/zbaylin/rofi-wifi-menu.
